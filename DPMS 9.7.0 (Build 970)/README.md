# DPMS - Discovery Package Manager (By The Discovery Team)

DPMS is a lightweight package manager designed for the **Discovery Linux project**.  
It comes with **CLI**, **TUI**, and **GUI** interfaces.

---

##Features

- Install, uninstall, and manage packages from local or remote repositories
- Has an Interactive GUI mode for people who are new to computers
- It is updated more often than other package managers
- It can even run on slow systems
- Can build tar.xz and tar.gz files of any folder on your computer
